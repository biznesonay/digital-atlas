# Digital Atlas - Цифровой атлас инновационной инфраструктуры

## Описание проекта
Интерактивная карта объектов инновационной инфраструктуры Казахстана.

## Технологии
- Frontend: Next.js 14, TypeScript, Material-UI, Google Maps
- Backend: Express.js, TypeScript, PostgreSQL, Prisma ORM
- Мультиязычность: RU/KZ/EN

## Установка и запуск
1. Клонировать репозиторий
2. Установить зависимости: `npm run install:all`
3. Настроить переменные окружения
4. Запустить: `npm run dev`

## Структура проекта
- `/frontend` - Next.js приложение
- `/backend` - Express.js API сервер
